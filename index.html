    User private info

    Private part of user's info is encrypted with file private-pk.pem

    Contact Kaspersky Socialization Team developers to receive the key and then use the following shell command to decrypt the private data.

    $ echo '' | base64 -d | openssl rsautl -decrypt -inkey private-pk.pem

    In production the 3d party system should provide its public key which will be linked with the "realm".
